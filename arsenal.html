What is Arsenal

Arsenal is a GunPVP server on the playerserver host CubedCraft. You can expect lots of gamemodes, maps, guns and fun. Even tho we are updating Arsenal right now you can still get updates from here.

When can I play Arsenal

We don't know this right now but we should be open soon, around the next month or so.

Where do I play Arsenal

You will be able to play Arsenal on mc.cubedcraft.com, Join via that IP and run the command '/join arsenal'. If the server is offline wait for it to boot and there you go. Or you can join via the IP arsenal.playerservers.com if its online.
